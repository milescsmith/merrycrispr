merrycrispr.off_target_scoring¶

merrycrispr.off_target_scoring(otrf, spacers_df, nuclease_info, off_target_score_threshold, off_target_count_threshold)¶

Calculate a cumulative off-target score for a protospacer

Parameters

  • otrf (str) – Path to the results from Bowtie

  • spacers_df (DataFrame) – Dataframe containing spacers. Format should be {'gene_name','feature_id','start','stop','strand','spacer'}

  • nuclease_info (str) – dictionary series with nuclease characteristics from nuclease_list.csv

  • off_target_score_threshold (int) – Total off-target score threshold beyond which a spacer is rejected. Ranges from 0 to 100.

  • off_target_count_threshold (int) – Number of potential mismatches that should be tolerated.

Returns

Return type

DataFrame matching the one passed to spacers_df containing off-target scores
