merrycrispr.match_seq¶

merrycrispr.match_seq(rec, sequences)¶

Given a feature in a GTF/GFF read in by gtfparse, match_seq() will extract the corresponding DNA sequence and create a new pyfaidx.Sequence object

Parameters

  • rec (Series) – Information for a feature (i.e. gene, exon, etc…). Requires the following indices: strand, gene_name, feature, strand, start, end, seq_hash

  • sequences (Sequence) – Object containing sequences to match against the positions in the index.

Return type

Sequence

Returns

  • Sequence object with annotation from rec and sequence information from

  • sequences.
