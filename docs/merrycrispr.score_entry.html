merrycrispr.score_entry¶

merrycrispr.score_entry(scoring_entry, **kwargs)¶

General function for assigning assigning an on-target score to a protospacer

Parameters

  • scoring_entry (dict) – Dictionary for a protospacer containing, at a minimum a “sequence” element.

  • kwargs (dict) – Additional arguments corresponding to the scoring “method” and “cutoff” score and queue “place”

Returns

Return type

typing.Dict[str, float]
