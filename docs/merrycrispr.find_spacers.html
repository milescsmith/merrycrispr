merrycrispr.find_spacers¶

merrycrispr.find_spacers(itemlist, nuclease_info, restriction_sites=None)¶

Find protospacers in a sequence for a given nuclease. The search region can be more expansive than just the PAM (for scoring purposes) and strand can be taken into account. find_spacers() will ignore sequences that have a poly(T) sequence, high GC content, or a motif matching given restriction nuclease sequence.

Parameters

  • itemlist (Fasta) – Parsed FASTA with sequences to examine for spacers

  • nuclease_info (dict) – Information for the nuclease to use. Required keys include start, end, strand, pam, spacer_regex

  • restriction_sites (List[str], optional (default: None)) – For a found spacer, ignore it if it contains the sequence for recognition by these restriction endonucleases.

Returns

Return type

DataFrame
